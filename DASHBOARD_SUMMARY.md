# Phase 4 Dashboard - Полный обзор

## Что построили

**Три-слойный аналитический дашборд** с полностью новой архитектурой, красивой визуализацией и интерактивными фильтрами.

## Структура дашборда

```
Streamlit App (dashboard.py)
├── 📈 Страница 1: ОБЗОР
│   ├─ Метрики: Cost, Conversions, CPA, Days
│   ├─ График: Daily Cost + Conversions (30 дней)
│   ├─ График: Daily CPA тренд
│   └─ Сводка инсайтов
│
├── 🎯 Страница 2: ИНСАЙТЫ (Level 2)
│   ├─ Фильтры: Тип (Good/Bad/All) + Segment Type
│   ├─ График: CPA Ratio (bar)
│   ├─ График: Cost vs Conversions (scatter)
│   └─ Таблица: 33 сегмента с метриками
│
└── 🚀 Страница 3: КАМПАНИИ (Level 3)
    ├─ Dropdown: Выбор сегмента
    ├─ Метрики: Cost, Conv, CPA по топ-10
    ├─ График: Cost по кампаниям
    ├─ График: CPA vs Conversions
    └─ Таблица: Топ-10 кампаний

         ↓ (все данные из PostgreSQL)
    
PostgreSQL DB
├── account_kpi (30 rows)
├── segment_trends_30d (33 rows)
└── campaign_insights_30d (0-300 rows)
```

## Ключевые особенности

### 1. Интерактивность
- Фильтры: По типу сегмента, классификации
- Hover: Точные значения при наведении
- Zoom/Pan: Для детального анализа графиков
- Кнопка "Обновить": Пересчёт данных

### 2. Визуализация
- **Цветовая схема**: Зелёный (хорошо) / Красный (плохо)
- **Graphq типы**: Bar, Scatter, Line, Area
- **Легенда**: Кликаемая для скрытия/показа данных
- **Hover-информация**: Все метрики при наведении

### 3. Производительность
- Кеширование на 5 минут
- Быстрая навигация между страницами
- Оптимальная память (Streamlit обрабатывает автоматически)

### 4. Интерфейс
- Полностью на русском
- Понятные иконки (📈, 🎯, 🚀 и т.д.)
- Боковая панель с информацией и кнопками

## Использование

### Запуск

```bash

cd /opt/phase4
bash start-dashboard.sh

# Способ 2: Напрямую
streamlit run /opt/phase4/ui/dashboard.py
```

### Адрес
```
http://localhost:8501
```

### Навигация
- Боковое меню: Выбор страницы
- Фильтры: На каждой странице
- Кнопка "Обновить": Очистить кеш

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Port 8501 занят | Используй `streamlit run --server.port 8502` |
| "Connection refused" | Проверь БД доступна, .env правильный |
| Пустые графики | Запустит level1_kpi.py и level2_trends.py |
| Медленно первый раз | Нормально - Streamlit инициализирует кеш |
| Данные не обновляются | Нажми кнопку "🔄 Обновить" |
