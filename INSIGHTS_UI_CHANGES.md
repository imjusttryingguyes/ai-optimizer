# Изменения UI инсайтов - Упрощение интерфейса

## Дата: 09 апреля 2026

### Проблема
Старый интерфейс был слишком сложным с 10+ типами инсайтов и громоздкими фильтрами.

### Решение
Переделан UI на странице `/insights` с трёхуровневой фильтрацией.

## Структура после изменений

### 1. Новые режимы просмотра (Настройки)
- **7 дней** - инсайты за неделю
- **30 дней** - инсайты за месяц  
- **Тренды** - сравнение 7 дней vs 30 дней

### 2. Типы инсайтов (зависят от режима)

#### Для режимов "7 дней" и "30 дней":
- **Точки роста** (📈) - типы: ACCOUNT_CPA_TREND_GOOD, SEGMENT_LADDER_TREND_GOOD, SEGMENT_LADDER_WINNER
- **Точки просадки** (📉) - все остальные типы (BAD, WASTE)

#### Для режима "Тренды":
- **Улучшение** (📈) - метрики 7д > 30д
- **Ухудшение** (📉) - метрики 7д <= 30д

### 3. Остальные фильтры
- **Аккаунт** - выбор из списка аккаунтов
- **Мин. серьезность** - Все, Medium (40+), High (60+), Critical (80+)

## Классификация типов инсайтов

### GROWTH_TYPES
```
- ACCOUNT_CPA_TREND_GOOD
- SEGMENT_LADDER_TREND_GOOD
- SEGMENT_LADDER_WINNER
```

### DECLINE_TYPES
```
- ACCOUNT_LEADS_TREND_BAD
- SEGMENT_COMBINATION_CPA_BAD
- SEGMENT_COMBINATION_TREND_BAD
- SEGMENT_LADDER_CPA_BAD
- SEGMENT_LADDER_TREND_BAD
- RSYA_WASTE
- SEGMENT_LADDER_WASTE
```

## Измененные файлы

### 1. `/opt/ai-optimizer/web/app.py`
- Добавлены константы: `GROWTH_TYPES`, `DECLINE_TYPES`
- Добавлена функция: `classify_insight_type()` - классификация типов
- Добавлена функция: `get_insights_by_period()` - получение инсайтов за период
- Добавлена функция: `get_trend_data()` - расчет трендов (7д vs 30д)
- Переделан route `/insights` с новыми параметрами:
  - `mode`: '7days', '30days', 'trends'
  - `category`: 'growth', 'decline', 'improvement', 'decline'

### 2. `/opt/ai-optimizer/web/templates/insights.html`
- Переделан UI с тремя колонками фильтров вместо четырех
- Кнопки-фильтры вместо select для "Тип"
- Новая таблица с колонками для трендов (когда mode=trends)
- Добавлена функция `updateFilters()` для синхронизации фильтров
- Улучшена читаемость через эмодзи

## Примеры URL

### 7-дневные инсайты (рост)
```
/insights?mode=7days&category=growth
```

### 30-дневные инсайты (просадки) с фильтром по аккаунту
```
/insights?mode=30days&category=decline&account_id=mmg-sz&severity_min=40
```

### Тренды (улучшения)
```
/insights?mode=trends&category=improvement
```

### Тренды (ухудшения) с критической серьезностью
```
/insights?mode=trends&category=decline&severity_min=80
```

## Навигация

- **На главной странице**: кнопка "Инсайты" ведет на `/insights`
- **На странице инсайтов**: кнопки "Настройки" для быстрого переключения режимов
- **Фильтры**: автоматически применяются при выборе
- **Очистка**: кнопка "Очистить фильтры" на странице
