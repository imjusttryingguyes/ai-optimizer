# 🚀 Как запустить дашборд (всё, что нужно)

## ✅ Быстрый старт (выбери один из способов)

### Способ 1: Автоматический launcher (рекомендуется)
```bash
bash /opt/phase4/start-dashboard-alt.sh
```
📊 Откроется на: **http://localhost:8502**

### Способ 2: Если Способ 1 не работает
```bash
cd /opt/phase4
streamlit run ui/dashboard.py --server.port 8503
```
📊 Откроется на: **http://localhost:8503**

### Способ 3: Стандартный (если нет занятых портов)
```bash
bash /opt/phase4/start-dashboard.sh
```
📊 Откроется на: **http://localhost:8501**

---

## 🎯 Что ты увидишь?

Откроется дашборд с **тремя страницами**:

### 📈 Страница 1: Обзор
- Ключевые метрики (Расход, Конверсии, CPA)
- Графики дневной динамики
- Сводка по инсайтам (хорошо/плохо)

### 🎯 Страница 2: Инсайты
- Таблица всех 33 сегментов
- Интерактивные фильтры
- Графики CPA и расходов
- Зелёные ✅ (хорошие) и красные ❌ (плохие)

### 🚀 Страница 3: Кампании
- Выбираешь сегмент
- Видишь топ-10 кампаний
- Их индивидуальные CPA и расходы

## 🔧 Если не работает

Смотри: `/opt/phase4/TROUBLESHOOTING.md`

**Самая частая проблема:**
> "Port 8501 is not available"

**Решение:**
```bash
bash /opt/phase4/start-dashboard-alt.sh  # На 8502
```

---

## 📝 Нужны свежие данные?

Запусти extraction скрипты:

```bash
# Уровень 1 (3 минуты)
python3 /opt/phase4/extraction/level1_kpi.py

# Уровень 2 (2 минуты)
python3 /opt/phase4/extraction/level2_trends.py

# Уровень 3 (10 минут, опционально)
python3 /opt/phase4/extraction/level3_campaign_30d.py
```

Потом нажми на дашборде кнопку **"🔄 Обновить"**

---

## 🎮 Навигация на дашборде

- **Боковое меню**: Выбор страницы
- **Фильтры**: На каждой странице (Тип сегмента, Классификация)
- **Графики**: Hover для деталей, Zoom/Pan для масштабирования
- **Кнопка обновить**: Очистить кеш и пересчитать
