# Streamlit KPI Dashboard

## Описание

Современный веб-дашборд для мониторинга KPI рекламных кампаний на базе **Streamlit**. Полностью интерактивный интерфейс без необходимости писать HTML/CSS/JavaScript.

## Преимущества Streamlit

✅ **Надёжность** - нет зависания, работает стабильно  
✅ **Простота** - чистый Python, легко поддерживать  
✅ **Скорость разработки** - быстрые итерации  
✅ **Встроенная интерактивность** - все компоненты из коробки  
✅ **Горячая перезагрузка** - изменения видны сразу при сохранении файла  

## Запуск дашборда

### Способ 1: Прямая команда
```bash
cd /opt/ai-optimizer
streamlit run dashboard_streamlit.py
```

### Способ 2: Скрипт
```bash
./start-streamlit-dashboard.sh
```

## Доступ

📍 **URL**: http://localhost:8501

Дашборд откроется автоматически в браузере.

## Функциональность

### 1️⃣ Выбор аккаунта
- Выпадающее меню слева для выбора рекламного аккаунта
- Автоматически загружаются только активные аккаунты с данными

### 2️⃣ Две вкладки

#### Вкладка "📈 KPI Статус"
Отображает текущие метрики:
- 💰 **Бюджет** - потрачено vs план, темп трат
- 📊 **Лиды** - количество лидов vs план, темп
- 💵 **CPA** - стоимость за лида vs план, отклонение
- ✅ **Статус** - общий статус (OK/Warning/Critical)

Прогноз на конец месяца:
- Ожидаемый расход
- Ожидаемое количество лидов
- Ожидаемый CPA

Алерты - важные замечания и рекомендации.

#### Вкладка "⚙️ Установить KPI"
Форма для установки плана на месяц:
- Месяц (автоматически текущий)
- Бюджет (₽)
- Целевое количество лидов
- Целевой CPA (₽)
- ROI Target (опционально, %)

После сохранения метрики автоматически обновляются.

## Структура файла

```python
dashboard_streamlit.py
├── get_db_connection()      # Подключение к PostgreSQL
├── get_accounts()           # Получить список аккаунтов
├── get_current_plan()       # Получить KPI план
├── save_plan()              # Сохранить KPI план
├── get_kpi_status()         # Получить статус KPI
└── UI элементы              # Streamlit компоненты
```

## Требования

- Python 3.9+
- PostgreSQL подключение (переменные окружения в .env)
- Пакеты: `streamlit`, `pandas`, `psycopg2`

## Окружение

Используются переменные окружения из `.env`:
```
DB_HOST=localhost
DB_PORT=5432
DB_NAME=ai_optimizer
DB_USER=postgres
DB_PASSWORD=password
```

## Горячая перезагрузка

При редактировании файла `dashboard_streamlit.py`, изменения видны сразу:
1. Сохраните файл
2. В браузере нажмите кнопку "Rerun" (появится вверху)
3. Дашборд перезагружается с новыми изменениями

## Остановка

Нажмите `Ctrl+C` в терминале где запущен Streamlit.

## Troubleshooting

**Q: Дашборд не загружается**  
A: Проверьте подключение к БД, переменные окружения в `.env`

**Q: Список аккаунтов пуст**  
A: Убедитесь, что в таблице `kpi_daily_summary` есть данные

**Q: Форма не сохраняет данные**  
A: Проверьте права доступа БД, таблица `kpi_monthly_plan` должна быть доступна для INSERT

**Q: Медленно загружается**  
A: Это нормально при первом запуске. Streamlit кэширует результаты.
