# AI Optimizer Roadmap

## Stage 1 — Foundation
- Data ingestion
- PostgreSQL storage
- KPI views
- Telegram digest
- Basic insight engine

## Stage 2 — Unified signals
- Campaign insights written to DB
- Segment insights written to DB
- Shared evidence format
- Shared priority model

## Stage 3 — Combination insights
- age × ad_type
- device × network
- campaign × device
- campaign × age
- geo × device
- weekday × hour

## Stage 4 — Recommendation engine
- exclude placement
- lower bids on segment
- split weak segment into separate campaign
- pause ineffective format
- detect data-quality issues

## Stage 5 — True optimizer
- recurring action suggestions
- optional semi-auto optimization
- portfolio view across many accounts
