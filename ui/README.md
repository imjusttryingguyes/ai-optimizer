# Phase 4 Dashboard

## Что это?

Современный Streamlit дашборд для визуализации всех трёх уровней аналитики Phase 4.

**Полностью новая архитектура** - ни одной строки старого кода, чистая разработка.

## Структура

### 📈 Страница "Обзор"
- Метрики: Расход, Конверсии, CPA, Дни данных
- График: Расход + Конверсии (30 дней)
- График: Тренд CPA по дням
- Сводка: Кол-во хороших/плохих/всего инсайтов

### 🎯 Страница "Инсайты" (Level 2)
- Фильтры: По типу (Хорошие/Плохие/Все) + По типу сегмента
- График 1: CPA Ratio (горизонтальный bar)
  - Зелёные = хорошо (≤0.67x)
  - Красные = плохо (≥1.5x)
- График 2: Scatter - Расход vs Конверсии
  - Размер точки = CPA
- Таблица: Все сегменты с метриками

### 🚀 Страница "Кампании" (Level 3)
- Dropdown: Выбор сегмента для drill-down
- Метрики: Расход, Конверсии, CPA по топ-10 кампаниям
- График 1: Расход по кампаниям
- График 2: CPA vs Конверсии
- Таблица: Топ-10 кампаний для сегмента

## Запуск

### Требования
```bash
pip install streamlit plotly pandas psycopg2-binary
```

### Просто запусти
```bash
cd /opt/phase4
bash start-dashboard.sh
```

Или прямо:
```bash
streamlit run ui/dashboard.py
```

Дашборд будет на `http://localhost:8501`

## Функции

✅ **Три страницы**
- Обзор (KPI + графики)
- Инсайты (Level 2 анализ)
- Кампании (Level 3 drill-down)

✅ **Интерактивные фильтры**
- По типу сегмента
- По классификации (хорошие/плохые)
- По выбранному сегменту для drill-down

✅ **Красивые графики**
- Plotly для интерактивности
- Цветовая схема: зелёный (хорошо) / красный (плохо)
- Hover-подсказки

✅ **Кеширование**
- Данные кешируются на 5 минут
- Кнопка "Обновить" чтобы пересчитать
- Быстро переключается между страницами

✅ **Русский интерфейс**
- Все описания и названия на русском
- Понятные иконки и структура

## Примеры использования

### 1. Быстрый обзор
Откройешь дашборд → Сразу видишь:
- 💰 Расход за 30 дней: 2,497,806₽
- 🎯 Конверсии: 362
- 📈 Средний CPA: 6,900₽

### 2. Анализ инсайтов
Переходишь на "Инсайты" →
- Видишь 14 хороших сегментов (✅ Хорошие)
- Видишь 19 плохих (❌ Проблемные)
- Кликаешь на сегмент, смотришь его метрики

Пример хорошего:
```
✅ Placement = mail.ru
   Расход: 266,801₽
   Конверсии: 61
   CPA: 4,374₽ (0.63x средней)
```

Пример плохого:
```
❌ CriterionType = RETARGETING
   Расход: 242,694₽
   Конверсии: 4
   CPA: 60,674₽ (8.79x средней!)
```

### 3. Drill-down в кампании
Переходишь на "Кампании" →
- Выбираешь интересующий сегмент (например, mail.ru)
- Смотришь, какие кампании там работают
- Видишь топ-10 кампаний по расходу с их CPA
- Можешь увидеть, какие кампании приносят прибыль (зелёные) и какие убыток (красные)

## Что передает дашборд

**От базы данных через API:**
1. account_kpi → 30 дневных метрик
2. segment_trends_30d → 33 инсайта
3. campaign_insights_30d → Drill-down по каждому сегменту

**Все графики интерактивные:**
- Hover показывает точные значения
- Zoom/Pan для детального просмотра
- Клик на легенду чтобы спрятать/показать данные

## Оптимизация

**Кеширование**: Данные кешируются на 5 минут
- Первый раз может быть медленно (~2 сек)
- Второй раз мгновенно
- Кнопка "Обновить" чтобы очистить кеш

**Легко масштабируется:**
- Добавить новую страницу: просто функция page_name()
- Добавить новый график: px.bar / go.Scatter
- Добавить фильтр: st.selectbox / st.radio

## Технические детали

- **Framework**: Streamlit (очень быстро для разработки)
- **Графики**: Plotly (интерактивные, красивые)
- **Данные**: Прямо из PostgreSQL
- **Кеширование**: st.cache_data
- **Язык**: Русский UI + Python
