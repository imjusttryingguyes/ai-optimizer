# 🌐 AI Optimizer Web Dashboard

Веб-дашборд для просмотра и анализа инсайтов AI Optimizer.

## Функции

- 📊 **Дашборд** - общая статистика и графики инсайтов
- 🔍 **Список инсайтов** - все инсайты с фильтрами по дате, типу, аккаунту, серьезности
- 📈 **Графики**:
  - Распределение инсайтов по типам (doughnut)
  - Распределение по уровню серьезности (bar)
  - Тренд инсайтов (line, последние 30 дней)
  - Топ-10 аккаунтов по воздействию (horizontal bar)
- ⚡ **Быстрые фильтры** - по периодам (7/14/30/90 дней)
- 📱 **Адаптивный дизайн** - работает на мобильных устройствах

## Установка

```bash
cd /opt/ai-optimizer/web
pip install -r requirements.txt
```

## Запуск

### Разработка (одна команда)
```bash
cd /opt/ai-optimizer/web
python app.py
```

Приложение будет доступно на `http://localhost:5000`

### Production (через скрипт)
```bash
./scripts/run_web_dashboard.sh
```

## API Endpoints

### Pages
- `GET /` - Главная страница дашборда
- `GET /insights` - Список всех инсайтов

### Data APIs
- `GET /api/insights-timeline?days=30` - Тренд инсайтов (JSON)
- `GET /api/top-accounts?days=7` - Топ аккаунты (JSON)

## Query Parameters

### Дашборд (`/`)
- `days=7|14|30` - период для анализа (по умолчанию 7)

### Инсайты (`/insights`)
- `days=7|14|30|90` - период поиска (по умолчанию 7)
- `type=TYPE_NAME` - фильтр по типу инсайта
- `account_id=ACCOUNT` - фильтр по аккаунту
- `severity_min=0|40|60|80` - минимум серьезности (по умолчанию 0)

## Примеры

```
# Дашборд за последние 30 дней
http://localhost:5000/?days=30

# Критичные инсайты (severity >= 80) за 14 дней
http://localhost:5000/insights?days=14&severity_min=80

# Все RSYA_WASTE инсайты
http://localhost:5000/insights?type=RSYA_WASTE

# Инсайты для конкретного аккаунта
http://localhost:5000/insights?account_id=mmg-sz
```

## Технологический стек

- **Backend**: Flask 3.1
- **Database**: PostgreSQL (через psycopg2)
- **Frontend**: Bootstrap 5, Chart.js 4
- **Server**: Python built-in development server

## Серьезность инсайтов

- 🔥 **Critical** (80+) - требует немедленного внимания
- 🔴 **High** (60-79) - важное предупреждение
- ⚠️ **Medium** (40-59) - рекомендация к действию
- ℹ️ **Low** (0-39) - информационное сообщение

## Производительность

- Автоматическое обновление дашборда каждые 5 минут
- Лимит вывода: 500 инсайтов на странице
- Индексированные запросы к БД для быстрого поиска
