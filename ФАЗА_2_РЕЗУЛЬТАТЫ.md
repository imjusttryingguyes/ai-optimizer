## Как использовать

### Одноразовая настройка
```bash
cd /opt/ai-optimizer
./start-daily-extraction.sh schedule
```
Готово! Выгрузка будет происходить ежедневно в 23:00.

### Ручная выгрузка
```bash
python3 ingestion/yandex_detailed_extract.py
```

### Проверить статус
```bash
./start-daily-extraction.sh check      # Показать статус cron
./start-daily-extraction.sh run        # Выполнить сейчас
./start-daily-extraction.sh stop       # Отключить cron
```

### Посмотреть логи
```bash
tail -f logs/daily_extract.log
```

## Известные проблемы

1. **Конверсии**: 0 конверсий получено
   - Причина: Проблема с форматом поля API
   - Статус: На расследовании

2. **Поисковые запросы**: Большей частью NULL
   - Причина: Тип отчета SEARCH_QUERY_REPORT не поддерживается
   - Решение: Можно получить через criterion_id

3. **Возраст/Пол**: Частичные данные
   - ~40% записей без информации (пользователи отказали от профилирования)
   - Достаточно для анализа трендов
